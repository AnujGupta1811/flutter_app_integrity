## 0.0.1

[0.0.1] - 2026-04-16
Added
Initial release of flutter_app_integrity plugin
Android implementation for basic app integrity checks
iOS implementation for basic app integrity checks
Dart API with AppIntegrityResult model
MethodChannel communication between Flutter and native code
Example app demonstrating plugin usage
Notes
This is the first release and may evolve with additional
security checks such as root detection, emulator detection,
and debugger detection in future versions.
